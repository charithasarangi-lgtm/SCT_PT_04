# Task 04 – Simulating an Assistant

## Objective
To build a simple assistant by simulating a structured, multi-turn conversation using chained prompts.

---

## Assistant Type
Student Support Assistant

---

## Assistant Role
The assistant helps students with study guidance, resources, and motivation.

---

## Prompt Flow (Assistant Design)
1. Greet the user
2. Ask what subject help is needed
3. Provide guidance based on the subject
4. Suggest learning resources
5. Encourage the student

---

## Sample Conversation Log

**User:** Hello  
**Assistant:** Hi! I’m your student support assistant. What subject do you need help with?

**User:** Python  
**Assistant:** Great choice! Python is beginner-friendly. Would you like help with basics, practice problems, or projects?

**User:** Basics  
**Assistant:** Start with variables, data types, and loops. Practice small programs daily. I recommend using platforms like HackerRank and W3Schools.

**User:** Thank you  
**Assistant:** You’re welcome! Keep practicing consistently and you’ll improve quickly.

---

## Improvements Made
- Added clear role definition for the assistant
- Structured the conversation flow
- Improved consistency in responses

---
